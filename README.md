# AI Virtual Learning Assistant

## Overview
This project is an AI-based learning assistant built using Flask.

The application allows students to ask questions and receive answers based on study materials.

## Features
• AI chatbot interface

• Question answering system

• Web interface using Flask

• Simple NLP based response system

## Technologies
Python

Flask

HTML

CSS

## How to Run

pip install -r requirements.txt

python app.py

Open browser

http://127.0.0.1:5000
